# ✅ Чеклист: Этап 1 — Privacy Policy

## 📋 Что создано

- ✅ `privacy-policy-en.md` — английская версия (Markdown)
- ✅ `privacy-policy-ru.md` — русская версия (Markdown)
- ✅ `privacy-policy-uk.md` — украинская версия (Markdown)
- ✅ `index.html` — HTML страница с переключателем языков
- ✅ `DEPLOYMENT.md` — инструкция по публикации
- ✅ `README.md` — описание файлов

## 🎯 Что нужно сделать ТЕБЕ

### 1. Заполнить контактные данные

Открой каждый файл и замени:

**В файлах `.md`:**
- `[YOUR_EMAIL]` → твой email (например: `support@swipeshot.app` или личный)
- `[YOUR_NAME or COMPANY_NAME]` → твое имя или название компании

**В файле `index.html`:**
- Найди `[YOUR_EMAIL]` (3 раза — для каждого языка)
- Найди `[ВАШ_EMAIL]` (русский)
- Найди `[ВАШ_EMAIL]` (украинский)
- Замени на свой email

- Найди `[YOUR_NAME or COMPANY_NAME]` (английский)
- Найди `[ВАШЕ_ИМЯ]` (русский)
- Найди `[ВАШЕ_ІМ'Я]` (украинский)
- Замени на свое имя

**Пример:**
```
Email: support@swipeshot.app
Developer: Ustym Kovalchuk
```

### 2. Проверить HTML локально

Открой файл `privacy-policy/index.html` в браузере:
1. Двойной клик по файлу
2. Проверь, что переключатель языков работает
3. Проверь, что все тексты отображаются правильно

### 3. Опубликовать на GitHub Pages

Следуй инструкции в файле `DEPLOYMENT.md`:

**Кратко:**
1. Создай репозиторий на GitHub (публичный)
2. Загрузи папку `privacy-policy/`
3. Включи GitHub Pages в Settings → Pages
4. Получи URL: `https://твой-username.github.io/название-репо/privacy-policy/`

### 4. Проверить опубликованную страницу

1. Открой URL в браузере
2. Проверь все 3 языка
3. Проверь на мобильном (должна быть адаптивной)
4. Скопируй URL — он понадобится для следующего шага

### 5. Обновить URL в приложении

1. Открой файл `constants/privacy.ts`
2. Замени `'https://example.com/privacy-policy'` на свой GitHub Pages URL
3. Сохрани файл
4. Перезапусти приложение (`npm run ios`)
5. Проверь кнопку "Privacy Policy" в Settings — должен открыться браузер

**Подробная инструкция:** `HOW-TO-UPDATE-URL.md`
