# 📄 Privacy Policy для SwipeShot

Этот каталог содержит Privacy Policy на 3 языках для публикации в App Store.

## 📋 Файлы

- `privacy-policy-en.md` — английская версия (основная для App Store)
- `privacy-policy-ru.md` — русская версия
- `privacy-policy-uk.md` — украинская версия
- `DEPLOYMENT.md` — инструкция по публикации на GitHub Pages

## 🚀 Что дальше?

### Шаг 1: Заполнить контактные данные

В каждом файле замени:
- `[YOUR_EMAIL]` → твой email (например: `support@swipeshot.app`)
- `[YOUR_NAME or COMPANY_NAME]` → твое имя или название компании

### Шаг 2: Опубликовать на GitHub Pages

Следуй инструкции в файле `DEPLOYMENT.md`

### Шаг 3: Получить URL

После публикации твой URL будет:
```
https://твой-username.github.io/swipeshot/privacy-policy-en.html
```

Этот URL нужно будет добавить в:
1. App Store Connect (Privacy URL)
2. Приложение SwipeShot (Settings → Privacy Policy)

## ✅ Что покрывает эта Privacy Policy

- ✅ GDPR (Европа)
- ✅ CCPA (США)
- ✅ App Store Requirements (Apple)
- ✅ Простой язык (без юридического жаргона)
- ✅ Объясняет, что данные НЕ собираются на сервер

## 📝 Обновления

При изменении функциональности приложения обнови:
1. Дату "Last Updated" в начале документа
2. Соответствующие разделы
3. Опубликуй обновленную версию на GitHub Pages
